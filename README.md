# School Management System (Java Web)

A full-stack Java Web-based School Management System designed to manage students, teachers, classes, and academic performance.
This application allows teachers to submit grades, automatically calculates student averages, and provides controlled access for different user roles (Student, Teacher, Administrator).

## Project Overview

This system aims to digitalize academic management within a school environment by providing

Role-based authentication

Grade management per class and subject

Automatic average calculation

Structured relational database

Clean MVC architecture

The project is built as a real-world learning experience focused on backend architecture, database modeling, and Java Web development.
